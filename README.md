## 🗂️ Folder Structure

The repository is organized to cover **multiple government exams**, each with a consistent and easy-to-follow structure:

```bash
Government-Exams-Resources/ 📂
├─ SEBI_Assistant_Manager_General/ 💹       # Folder for SEBI exam resources
│   ├─ Official/ 📝                         # Official links, notifications, etc.
│   │   └─ Links.txt 🔗                     # Important links: results, admit cards, forms
│   ├─ Notes/ 🗒️                            # Study notes, organized by subject
│   └─ Previous_Year_Questions/ 🏛️          # Past exam papers, organized per year
├─ MSTC_MT_Personnel_Administration/ ⚙️     # Same folder structure
├─ NABARD_Assitant_Manager_CS_IT/ 🌾        # Same folder structure
```
