## 🖥️ 1. Computer Fundamentals

### 📌 What is a Computer?

A **computer** is an electronic device that **accepts data**, **processes it**, **stores it**, and **produces output**.

### 🧠 IPO Cycle

➡️ **Input → Process → Output**

### ✏️ Example

* Input: Keyboard
* Process: CPU
* Output: Monitor

---

## 🧩 2. Hardware vs Software

### 🔩 Hardware

Physical components you can **see and touch**.

**Examples:**

* Keyboard ⌨️
* Mouse 🖱️
* CPU
* Monitor 🖥️

### 💿 Software

Programs that tell hardware **what to do**.

**Examples:**

* Windows
* MS Word
* Chrome

📝 **Shortcut:**

> Hardware = Body | Software = Brain
