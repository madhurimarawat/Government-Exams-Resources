## <a name="documents"></a> 📄 Documents & Spreadsheets

**Description:** Files used for text documents, reports, presentations, and structured data storage.

This section includes file types like `.txt`, `.md`, `.docx`, `.pdf`, `.xlsx`, `.csv`, `.pptx`, `.odt`, `.rtf`, `.latex`, etc.

| **Extension** | **Description** 📖            | **Use Cases** 📃               | **Tools & Technologies** 🔧                   |
| ------------- | ---------------------------- | ----------------------------- | -------------------------------------------- |
| `.txt`        | Plain text file 📄            | Notes, logs, configs 🗒️        | Notepad++ 📝, VS Code 💻, Nano 📟               |
| `.md`         | Markdown file ✍️              | Documentation, READMEs 📖      | GitHub 🐙, Obsidian 🏗️, Jekyll 🌐               |
| `.docx`       | Microsoft Word file 🏛️        | Reports, essays, documents 📄  | Microsoft Word 🖊️, Google Docs 🌍              |
| `.doc`        | Older Word document 🏛️        | Legacy MS Word documents 📜    | Microsoft Word 🖊️, LibreOffice 🏢              |
| `.odt`        | OpenDocument Text 📜          | Open-source Word files 🏗️      | LibreOffice Writer 🖊️, OpenOffice 📖           |
| `.pdf`        | Portable Document Format 📜   | Official documents, e-books 📚 | Adobe Acrobat 🔍, Foxit Reader 📖              |
| `.rtf`        | Rich Text Format 📝           | Formatted text documents ✍️    | Microsoft Word 🖊️, WordPad 📜                  |
| `.tex`        | LaTeX file 📑                 | Academic papers, math docs 📖  | TeX Live 📚, Overleaf ✏️, MiKTeX 🏗️             |
| `.wpd`        | WordPerfect Document 🏛️       | Older word processing docs 📄  | WordPerfect 🖊️, LibreOffice 🏢                 |
| `.xlsx`       | Excel Spreadsheet 📊          | Data analysis, finance 💰      | Excel 📈, Google Sheets 📊, LibreOffice Calc 🏢 |
| `.xls`        | Legacy Excel file 📊          | Older Excel spreadsheets 🗃️    | Excel 📈, LibreOffice Calc 🏢                  |
| `.csv`        | Comma-Separated Values 📑     | Storing structured data 📊     | Pandas 🐼, Excel 📈, CSVKit 🛠️, OpenRefine 🔍    |
| `.tsv`        | Tab-Separated Values 📄       | Tabular data storage 🗃️        | Pandas 🐼, Excel 📈, Notepad++ 📝               |
| `.pptx`       | PowerPoint Presentation 🎤    | Slideshows, presentations 📽️   | PowerPoint 📊, Google Slides 🌍                |
| `.ppt`        | Older PowerPoint file 🎞️      | Legacy slideshows 📽️           | PowerPoint 📊, LibreOffice Impress 🏢          |
| `.odp`        | OpenDocument Presentation 🎥  | Open-source slideshows 📽️      | LibreOffice Impress 🏢, OpenOffice 📖          |
| `.pages`      | Apple Pages Document 🍏       | Apple document processing 📝   | Apple Pages 📖                                |
| `.key`        | Apple Keynote Presentation 🍏 | Apple slideshows 🎤            | Apple Keynote 📊                              |
| `.numbers`    | Apple Numbers Spreadsheet 🍏  | Apple spreadsheet files 📈     | Apple Numbers 📊                              |
| `.log`        | Log file 📜                   | System/application logs 🖥️     | Notepad++ 📝, VS Code 💻, Logstash 🔍           |

### **📌 Notes:**

- **Use `.txt` for simple notes** and `.md` for documentation.
- **Use `.docx`, `.odt`, or `.pdf`** for professional reports.
- **For spreadsheets, `.xlsx`, `.csv`, and `.tsv`** are most common.
- **Apple-specific formats (`.pages`, `.key`, `.numbers`)** work best in macOS/iOS.

## <a name="programming-code-files"></a> 💻 Programming & Code Files

Includes .py, .java, .c, .cpp, .html, .css, .js, .rb, .swift, .ts, .dart, .r, .scala, .pl, etc.

| **Extension** | **Description** 💾   | **Best For** 🎯                | **Tools & Technologies** 🛠️   |
| ------------- | ------------------- | ----------------------------- | ---------------------------- |
| `.py`         | Python script 🐍     | AI, automation, backend       | PyCharm, Jupyter, VS Code    |
| `.java`       | Java source file ☕  | Enterprise apps, Android dev  | IntelliJ, Eclipse, NetBeans  |
| `.c`          | C source file 🔵     | Low-level programming, OS dev | GCC, Clang, CodeBlocks       |
| `.cpp`        | C++ source file 🚀   | High-performance apps, gaming | Visual Studio, CMake         |
| `.html`       | HTML file 🌍         | Structuring web pages         | VS Code, Chrome DevTools     |
| `.css`        | CSS file 🎨          | Styling web pages             | Tailwind, Bootstrap, Sass    |
| `.js`         | JavaScript file ⚡   | Frontend & backend scripting  | Node.js, React, Angular      |
| `.ts`         | TypeScript file 🔵   | Safer JavaScript              | VS Code, Angular, Deno       |
| `.r`          | R script 📊          | Data science, statistics      | RStudio, Jupyter Notebook    |
| `.scala`      | Scala source file 🔴 | Scalable applications         | IntelliJ, Scala IDE          |
| `.dart`       | Dart source file 🎯  | Flutter mobile apps           | Dart SDK, Flutter            |
| `.rb`         | Ruby script 💎       | Web development, scripting    | RubyMine, VS Code            |
| `.swift`      | Swift source file 🍏 | iOS/macOS development         | Xcode, Swift Playgrounds     |
| `.go`         | Go source file 🏎️    | Cloud computing, networking   | GoLand, VS Code, Go Compiler |
| `.rs`         | Rust source file 🦀  | Systems programming, safety   | Rust Compiler, Cargo, Clippy |
| `.lua`        | Lua script 🕹️        | Game development, scripting   | Lua, Love2D, Corona SDK      |
| `.pl`         | Perl script 🐪       | Text processing, sys admin    | Perl, Strawberry Perl, Padre |
| `.sh`         | Shell script 🖥️      | Linux automation              | Bash, Zsh, Terminal          |
| `.bat`        | Batch script 📝      | Windows automation            | CMD, PowerShell              |

**📌 Notes:**

- `.py` is widely used for automation 🤖, AI 🤯, and backend development.
- `.java` powers enterprise applications 🏢 and Android development 📱.
- `.cpp` is preferred for high-performance applications 🚀 like gaming and system software.
- `.sh` and `.bat` automate system tasks 🛠️, with `.sh` for Linux/macOS and `.bat` for Windows.

## <a name="web-graphics"></a> 🌐 Web & Graphics

**Description:** This section covers popular image formats used for the web, UI design, and digital graphics. Formats vary in compression, scalability, and transparency support.

Covers `.png`, `.jpg`, `.svg`, `.gif`, `.webp`, `.ico`, `.tiff`, `.bmp`, etc.

| **Extension** | **Description** 🖼️          | **Best For** 🎯              | **Tools & Technologies** 🛠️ |
| ------------- | -------------------------- | --------------------------- | -------------------------- |
| `.png`        | High-quality images 🌆      | Transparency, UI elements   | Photoshop, GIMP, Figma     |
| `.jpg`        | Compressed images 📷        | Photographs, web graphics   | Photoshop, Lightroom       |
| `.webp`       | Web-optimized images 🌎     | Fast-loading web graphics   | Squoosh, WebP Converter    |
| `.svg`        | Scalable vector graphics 🎨 | Icons, logos                | Illustrator, Inkscape      |
| `.gif`        | Animated images 🎞️          | Simple animations, memes    | Photoshop, GIMP, Ezgif     |
| `.ico`        | Icon file 🏠                | Website favicons, app icons | IcoFX, Photoshop, GIMP     |
| `.tiff`       | Lossless raster images 🖼️   | High-quality prints         | Photoshop, Lightroom       |
| `.bmp`        | Bitmap image 📏             | Uncompressed image storage  | MS Paint, Photoshop        |
| `.psd`        | Photoshop document 🎭       | Layered image editing       | Adobe Photoshop            |
| `.ai`         | Adobe Illustrator file 🖋️   | Vector graphics, logos      | Adobe Illustrator          |

### **📌 Notes:**

- `.png` supports transparency 🎭, making it ideal for UI elements and logos.
- `.jpg` uses lossy compression 📷, best for high-quality photos with small file sizes.
- `.webp` is optimized for web performance 🚀, offering better compression than PNG and JPG.
- `.svg` is resolution-independent 📏, perfect for icons and scalable graphics.

## <a name="configuration-system-files"></a> ⚙️ Configuration & System Files

**Description:** Used for configuring applications, defining settings, and automating system processes.

Includes `.ini`, `.cfg`, `.toml`, `.yaml`, `.sh`, `.bat`, `.conf`, `.editorconfig`, etc.

| **Extension**   | **Description** ⚙️     | **Best For** 🎯            | **Tools & Technologies** 🛠️  |
| --------------- | --------------------- | ------------------------- | --------------------------- |
| `.ini`          | Initialization file 🛠️ | App settings, preferences | Notepad++, VS Code          |
| `.cfg`          | Configuration file ⚙️  | Software & system configs | Any text editor             |
| `.toml`         | TOML config file 📜    | Readable configuration    | VS Code, Sublime Text       |
| `.yaml`         | YAML config file 📄    | Human-readable settings   | PyYAML, Kubernetes, Ansible |
| `.json`         | JSON settings file 🔄  | APIs, web configs         | VS Code, Postman, jq        |
| `.xml`          | XML configuration 📜   | Web & software settings   | Notepad++, XML Notepad      |
| `.sh`           | Shell script 🖥️        | Linux/macOS automation    | Bash, Zsh, Terminal         |
| `.bat`          | Batch script 📝        | Windows automation        | CMD, PowerShell             |
| `.conf`         | System config file ⚙️  | Linux system settings     | Vim, Nano, VS Code          |
| `.editorconfig` | Code style settings 🖋️ | Formatting rules          | EditorConfig Plugin         |

### **📌 Notes:**

- `.ini` and `.cfg` are commonly used for app configurations ⚙️.
- `.toml` and `.yaml` offer structured, human-readable settings 📄.
- `.json` is widely used for API settings and web applications 🔄.
- `.sh` automates tasks in Linux/macOS 🖥️, while `.bat` does the same for Windows 📝.

## <a name="media-files"></a> 🎧🎥 Media Files

**Description:** Used for storing, streaming, and playing audio and video content in various formats.

Includes `.mp4`, `.mp3`, `.wav`, `.flac`, `.mov`, `.avi`, `.mkv`, `.webm`, `.aac`, `.ogg`, `.m4a`, `.flv`, `.3gp`, `.ts`, etc.

| **Extension** | **Description** 🎬🎵      | **Best For** 🎯             | **Tools & Technologies** 🛠️      |
| ------------- | ----------------------- | -------------------------- | ------------------------------- |
| `.mp4`        | MP4 Video File 📽️        | Streaming, online videos   | VLC, FFmpeg, Adobe Premiere Pro |
| `.mp3`        | MP3 Audio File 🎵        | Music, podcasts            | VLC, Audacity, iTunes           |
| `.wav`        | WAV Audio File 🎼        | High-quality sound         | Audacity, FL Studio, Logic Pro  |
| `.flac`       | FLAC Audio File 🎶       | Lossless music storage     | VLC, Foobar2000, Audacity       |
| `.aac`        | AAC Audio File 🎙️        | Apple music format         | iTunes, QuickTime, Audacity     |
| `.ogg`        | Ogg Vorbis Audio 🎧      | Open-source alternative    | VLC, Audacity                   |
| `.wma`        | Windows Media Audio 🔊   | Windows music playback     | Windows Media Player, VLC       |
| `.m4a`        | M4A Audio File 🎵        | iTunes music format        | Apple Music, QuickTime          |
| `.opus`       | Opus Audio Format 🎤     | Voice recording, streaming | Discord, WhatsApp, FFmpeg       |
| `.mov`        | QuickTime Movie 🎬       | Apple video format         | QuickTime, Final Cut Pro        |
| `.avi`        | AVI Video File 📹        | Windows video format       | VLC, Windows Media Player       |
| `.mkv`        | Matroska Video File 🎞️   | High-quality HD videos     | VLC, MKVToolNix                 |
| `.webm`       | WebM Video File 🌍       | Web streaming format       | YouTube, FFmpeg, Chrome         |
| `.flv`        | Flash Video File 📺      | Older web videos           | Adobe Flash Player (deprecated) |
| `.3gp`        | 3GPP Video File 📱       | Mobile videos              | VLC, QuickTime                  |
| `.ts`         | MPEG Transport Stream 📡 | Broadcast, IPTV            | VLC, FFmpeg                     |
| `.rm`         | RealMedia File 🎥        | Older streaming format     | RealPlayer                      |
| `.m4v`        | Apple Video Format 🍏    | iTunes video files         | QuickTime, iTunes               |

### **📌 Notes:**

- `.mp4` is the most widely supported video format 📽️, ideal for streaming and downloads.
- `.mp3` is the standard for compressed audio 🎵, while `.flac` provides lossless quality.
- `.mkv` supports multiple audio and subtitle tracks 🎞️, great for HD videos.
- `.webm` is optimized for web streaming 🌍, commonly used on YouTube.

## <a name="database-files"></a> 💾 Database Files

**Description:** Files used for storing, managing, and backing up structured and unstructured database data.

Includes `.db`, `.sqlite`, `.sql`, `.mdb`, `.accdb`, `.frm`, `.ibd`, `.myd`, `.mdf`, `.ldf`, `.dmp`, `.bak`, `.json`, `.bson`, `.cql`, `.rdb`, `.rocksdb`, `.dbf`, `.kdbx`, etc.

| **Extension** | **Description** 🏛️       | **Best For** 🎯                  | **Tools & Technologies** 🛠️  |
| ------------- | ----------------------- | ------------------------------- | --------------------------- |
| `.db`         | Generic database file 📂 | Storing structured data         | SQLite, DB Browser          |
| `.sqlite`     | SQLite database 📀       | Lightweight database            | SQLite CLI, DB Browser      |
| `.sql`        | SQL script file 📝       | Database queries, schema        | MySQL, PostgreSQL, Oracle   |
| `.mdb`        | Microsoft Access DB 📊   | Windows-based database          | MS Access, MDB Viewer       |
| `.accdb`      | Access Database (New) 📑 | MS Access newer format          | MS Access                   |
| `.frm`        | MySQL Table Format 📜    | MySQL table definitions         | MySQL, MariaDB              |
| `.ibd`        | InnoDB Storage File 🏗️   | MySQL InnoDB tables             | MySQL, InnoDB               |
| `.myd`        | MyISAM Data File 📊      | MySQL table storage             | MySQL, MariaDB              |
| `.myi`        | MyISAM Index File 🔎     | MySQL table indexes             | MySQL, MariaDB              |
| `.ndf`        | SQL Server Data File 🏢  | Secondary DB storage            | Microsoft SQL Server        |
| `.mdf`        | SQL Server Master DB 📂  | Primary database file           | Microsoft SQL Server        |
| `.ldf`        | SQL Server Log File 📝   | Transaction logging             | Microsoft SQL Server        |
| `.dmp`        | Database Dump 📦         | Database backups                | Oracle, MySQL, PostgreSQL   |
| `.bak`        | Backup File 🔄           | Database restoration            | Microsoft SQL Server, MySQL |
| `.tar.gz`     | Compressed DB Backup 📁  | Database migration              | PostgreSQL, MongoDB         |
| `.json`       | NoSQL Data Storage 🌐    | Document-based databases        | MongoDB, Firebase           |
| `.bson`       | Binary JSON Format 📦    | MongoDB storage format          | MongoDB                     |
| `.cql`        | Cassandra Query File 📜  | NoSQL distributed DB            | Apache Cassandra            |
| `.rdb`        | Redis Database Dump 🔥   | In-memory key-value store       | Redis                       |
| `.rocksdb`    | RocksDB Storage 📊       | High-performance storage engine | RocksDB                     |
| `.dbf`        | dBase Database 📋        | Legacy database files           | FoxPro, dBase               |
| `.kdbx`       | KeePass Database 🔐      | Encrypted password storage      | KeePass, KeePassXC          |

### **📌 Notes:**

- `.sqlite` and `.db` are lightweight database formats 📂, ideal for small applications.
- `.sql`, `.dmp`, and `.bak` store database queries and backups 📝 for migrations and recovery.
- `.json` and `.bson` are used in NoSQL databases 🌐, such as MongoDB and Firebase.
- `.rdb` and `.rocksdb` are optimized for fast in-memory storage 🔥.

## <a name="archives-compression"></a> 📀 Archives & Compression

**Description:** Compressed file formats used for reducing storage size, bundling multiple files, and packaging software.

Includes `.zip`, `.tar`, `.gz`, `.bz2`, `.7z`, `.rar`, `.xz`, `.iso`, `.vhd`, `.dmg`, `.img`, `.pkg`, `.appimage`, etc.

| **Extension** | **Description** 📦           | **Best For** 🎯                      | **Tools & Technologies** 🛠️           |
| ------------- | --------------------------- | ----------------------------------- | ------------------------------------ |
| `.zip`        | ZIP compressed file 📦       | Storing multiple files              | WinRAR, 7-Zip, macOS Archive Utility |
| `.tar`        | TAR archive 🎞️               | Linux backups                       | tar, gzip                            |
| `.gz`         | Gzip compression 🗜️          | Web assets, logs                    | gzip, gunzip                         |
| `.7z`         | 7-Zip compression 📂         | High-ratio compression              | 7-Zip, WinRAR                        |
| `.rar`        | RAR archive 📚               | File compression & storage          | WinRAR, PeaZip                       |
| `.bz2`        | Bzip2 compression 📉         | Higher compression efficiency       | bzip2, WinRAR                        |
| `.xz`         | XZ compression 🚀            | Efficient Linux compression         | xz, tar.xz                           |
| `.tar.gz`     | Gzipped TAR archive 🎞️       | Combining multiple compressed files | tar, gzip                            |
| `.tar.bz2`    | Bzip2-compressed TAR 📚      | High-compression Linux archive      | tar, bzip2                           |
| `.tar.xz`     | XZ-compressed TAR 🚀         | Maximum compression                 | tar, xz                              |
| `.lzma`       | LZMA compression 🔄          | High-ratio compression              | 7-Zip, LZMA Utils                    |
| `.zst`        | Zstandard compression ⚡     | Fast compression with small size    | zstd, tar.zst                        |
| `.cab`        | Cabinet archive 📂           | Windows installation files          | Windows Installer, cabextract        |
| `.iso`        | ISO disk image 💿            | Bootable OS images                  | Rufus, VirtualBox                    |
| `.vhd`        | Virtual Hard Disk 💾         | Windows virtual machines            | Hyper-V, VirtualBox                  |
| `.dmg`        | macOS Disk Image 🍏          | macOS app distribution              | Disk Utility, TransMac               |
| `.img`        | Raw disk image 💽            | System backups, embedded OS         | Win32 Disk Imager, Etcher            |
| `.pkg`        | macOS Installer Package 📦   | macOS software installation         | Installer.app                        |
| `.xpi`        | Mozilla Extension Archive 🦊 | Firefox & Thunderbird add-ons       | Mozilla Firefox                      |
| `.appimage`   | Linux AppImage 🔗            | Portable Linux applications         | AppImageLauncher                     |

### **📌 Notes:**

- `.zip` and `.rar` are common for file compression and storage 📂.
- `.tar.gz`, `.tar.bz2`, and `.tar.xz` are widely used in Linux systems 🎞️.
- `.iso` and `.img` store disk images for OS installations 💿.
- `.appimage` provides a portable application format for Linux 🔗.
