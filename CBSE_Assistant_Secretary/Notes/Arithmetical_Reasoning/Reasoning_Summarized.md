## 🔢 1. Series (Number & Alphabet)

### 📌 Types Covered

* Missing Number
* Wrong Term
* Alphabet Series

### 🧠 How to Approach

1. Check **difference** between numbers
2. Check **alternating patterns**
3. Look for **multiplication / division**
4. For alphabets → convert letters to **positions (A=1, B=2 … Z=26)**

### 🧮 Common Patterns

* +2, +4, +6 … (Increasing difference)
* ×2, ×3
* Square / Cube
* Alternating (+ / −)

### ✏️ Example

**Q:** 2, 6, 12, 20, ?

**Solution:**

* Differences: +4, +6, +8
* Next = +10
* ✅ Answer = **30**

## 🔗 2. Analogy

### 📌 Types

* Number Analogy
* Word Analogy
* Alphabet Position Analogy

### 🧠 Formula

> A : B :: C : ?

👉 Relationship between A & B = same for C

### ✏️ Number Analogy Example

**Q:** 4 : 16 :: 5 : ?

**Logic:**

* 4 × 4 = 16
* 5 × 5 = **25** ✅

### ✏️ Alphabet Analogy Example

**Q:** C : F :: E : ?

**Logic:**

* C → F (+3)
* E → H (+3)
* ✅ Answer = **H**

## 🔐 3. Coding–Decoding

### 📌 Types

* Letter Shifting
* Word Coding
* Simple Number Coding

### 🧠 Letter Shifting Rule

* Forward shift: A → B → C
* Backward shift: Z → Y → X

### ✏️ Example

**Q:** If CAT is coded as DBU, then DOG is coded as?

**Logic:**

* C → D (+1)
* A → B (+1)
* T → U (+1)

DOG → D→E, O→P, G→H

✅ Answer = **EPH**

## 👨‍👩‍👧‍👦 4. Blood Relations

### 📌 Common Relations

* Father / Mother
* Brother / Sister
* Son / Daughter
* Uncle / Aunt

### 🧠 Golden Rule

📝 **Always draw a quick family tree**

### ✏️ Example

**Q:** A is the brother of B. B is the mother of C. How is A related to C?

**Solution:**

* A = Brother of B
* B = Mother of C
* So A = **Maternal Uncle** ✅

## 🧭 5. Direction Sense

### 📌 Directions

* North (↑)
* South (↓)
* East (→)
* West (←)

### 🧠 Trick

👉 Always **draw arrows** and track movement step by step

### ✏️ Example

**Q:** A person moves 5 km north, then 3 km east. How far is he from start?

**Solution:**

* Use Pythagoras
* √(5² + 3²) = √34

✅ Answer = **√34 km**

## 🔵 6. Syllogism (Basic)

### 📌 Statements Used

* All A are B
* Some A are B
* No A is B

### 🧠 Rule

🟢 Draw **Venn diagrams** mentally
❌ Avoid assumptions

### ✏️ Example

**Statements:**

* All cats are animals
* Some animals are dogs

**Conclusion:** Some cats are dogs ❌ (Not definite)
